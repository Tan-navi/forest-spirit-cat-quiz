# 🌿 Forest Spirit Cat Quiz 🐱

A cozy forest-themed personality quiz built with Flask, HTML, CSS and JavaScript.

Users answer a series of personality questions and discover which magical forest spirit cat matches their personality.

## Features

- Interactive personality quiz
- Four unique forest spirit cats
- Custom illustrations
- Background music and sound effects
- Animated UI
- Flask-powered website

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
